## 🗺️ Master System Blueprint: 레이아웃 와이어프레임 (Figma 실행 계획)

**목표:** 시스템 설계 방법론의 신뢰도와 미래지향적인 이미지를 시각적으로 완벽히 구현하여, 잠재 고객에게 '시스템 설계자'로서의 권위를 즉각 전달한다.

**전체 스타일 적용 규칙:**
*   **컬러 팔레트:** 주조색 (Deep Blue: `#0A1931`), 강조색 (Electric Purple: `#BB86FC`), 보조색 (Neon Cyan: `#00FFFF`, Light Gray: `#F0F0F0`).
*   **타이포그래피:** 제목(Headlines)은 `Orbitron` 사용, 본문(Body Text)은 `Inter` 사용.
*   **레이아웃 컨셉:** 모듈식 그리드 기반, 깊이감(Depth)과 미래적인 연결성(Connectivity) 강조.

### 1. 인트로 섹션 (The Architect's Statement)

| 요소 | 내용/스타일링 지침 | 상세 구현 (Figma Layering) |
| :--- | :--- | :--- |
| **배경** | Deep Blue (`#0A1931`) 배경에 은은한 그라데이션 또는 미세한 그리드 패턴 적용. | 전체 배경에 낮은 대비의 네온 라인(Neon Cyan)을 미세하게 삽입하여 시스템적 느낌 부여. |
| **메인 타이틀** | **"Master System Blueprint"** | 폰트: Orbitron, 크기: 72pt 이상, 색상: Electric Purple (`#BB86FC`). 중앙 정렬. |
| **서브 헤드라인** | "AI 시스템을 설계하는 궁극의 방법론" | 폰트: Inter, 색상: Light Gray (`#F0F0F0`). 메인 타이틀 아래에 배치. |
| **비주얼 요소** | 중앙에 추상적인 데이터 흐름 또는 신경망(Neural Network) 형태의 모션 그래픽/아이콘 사용. | Deep Blue와 Electric Purple을 활용한 빛나는 연결선(네온 효과)으로 구성된 다이어그램 삽입. |

### 2. 방법론 섹션 (The Methodology Flow)

**목표:** 복잡한 시스템 설계 프로세스를 논리적이고 시각적으로 명확하게 제시하여 신뢰도를 확보한다.

| 요소 | 내용/스타일링 지침 | 상세 구현 (Figma Layering) |
| :--- | :--- | :--- |
| **구조** | 4단계 또는 5단계의 선형적(Linear) 혹은 순환적(Cyclical) 프로세스 다이어그램. | 좌측 또는 중앙에 수직으로 배치하여 흐름을 강조. |
| **모듈 박스** | 각 단계별 모듈을 독립된 카드 형태로 구성. | 배경색: 약간 밝은 Deep Blue (`#152A40`). 테두리(Stroke)는 Electric Purple로 설정. |
| **흐름 연결선** | 각 모듈 간의 연결은 화살표가 아닌, 빛나는 에너지 흐름(Neon Cyan)으로 표현하여 '시스템 작동'의 느낌 강조. | 선의 두께와 밝기를 조절하여 시각적 계층 구조를 명확히 함. |
| **텍스트** | 각 단계 설명은 간결하게 작성하고, 핵심 키워드는 Orbitron으로 강조. | 본문 텍스트는 Inter로 유지하며, 프로세스 흐름을 따라 자연스럽게 읽히도록 배치. |

### 3. 기능 블록 섹션 (Feature Block & ROI Visualization)

**목표:** 시스템이 제공하는 구체적인 가치와 투자 대비 수익(ROI)을 시각적으로 증명한다.

| 요소 | 내용/스타일링 지침 | 상세 구현 (Figma Layering) |
| :--- | :--- | :--- |
| **레이아웃** | 3~4개의 핵심 기능 블록과 그 아래에 ROI 검증 섹션을 분리. | 그리드 시스템을 활용하여 좌측(기능), 우측(결과/수익성)으로 이분화된 레이아웃 적용. |
| **기능 카드** | 각 기능은 아이콘 + 제목 + 간략한 설명으로 구성. | 기능 박스는 Deep Blue 배경에 Electric Purple 강조선 적용. |
| **ROI 시각화** | 핵심 KPI(Key Performance Indicators)를 막대 그래프 또는 원형 차트 형태로 배치. | ROI 데이터는 Neon Cyan과 Electric Purple을 사용하여 '수익성'을 극적으로 부각. (예: 수익성이 높은 부분은 밝고 선명하게). |
| **CTA** | 최종 행동 유도 버튼은 Deep Blue 배경에 Electric Purple 텍스트로 설정하여 시선 집중. | CTA 버튼은 시스템의 신뢰감을 바탕으로 명확한 다음 단계(Next Step)를 제시하도록 디자인. |
