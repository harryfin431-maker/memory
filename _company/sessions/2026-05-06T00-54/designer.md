## 🎬 유튜브 영상 인트로 및 핵심 슬라이드 디자인 파일 (Mockup Specification)

확정된 **Deep Blue/Neon 비주얼 시스템 가이드라인**을 기반으로, MVP 콘텐츠에 즉시 적용 가능한 최종 디자인 사양(Specification)을 제공합니다. 이 사양은 Figma 등으로의 직접적인 구현을 위한 좌표와 코드를 포함하고 있습니다.

### 1. 영상 인트로 화면 (Intro Screen Mockup)

**목표:** 시청자의 주의를 즉각적으로 사로잡고, '시스템'이라는 고가치 주제에 대한 신뢰감을 부여합니다.

| 요소 | 상세 사양 | 적용 스타일 |
| :--- | :--- | :--- |
| **배경 컬러** | `#0A1931` (Deep Blue - 배경) | Deep Blue |
| **강조 컬러** | `#00FFFF` (Neon Cyan - 하이라이트 및 라인) | Neon Cyan |
| **타이포그래피** | **Orbitron Bold** | 제목/핵심 문구용 |
| **텍스트 1 (후크)** | "노력은 시스템이 정한다." | Orbitron Bold, 크기: 60pt, 컬러: `#00FFFF` |
| **텍스트 2 (메인)** | "당신은 이제 수익을 설계하는 시스템 설계자다." | Inter Bold, 크기: 35pt, 컬러: `#FFFFFF` |
| **시각적 요소** | 화면 중앙에 복잡하게 얽힌 추상적인 선과 다이어그램이 빠르게 회전하며 지나감. 이 선들은 Neon Cyan으로 표현되어야 함. | Neon Cyan 라인 효과 (Motion Graphics) |
| **로고 배치** | 하단 우측에 '아 빨랑~~' 로고를 최소화된 흰색으로 배치하여 신뢰감을 유지. | White |

### 2. 핵심 슬라이드 디자인 가이드 (Core Slide Template)

**목표:** 시스템 설계 방법론의 논리적 흐름을 명확하고 미래지향적으로 전달합니다.

#### A. 타이틀 슬라이드 (Title Slide: System Framework)

| 요소 | 상세 사양 | 적용 스타일 |
| :--- | :--- | :--- |
| **배경 컬러** | `#0A1931` (Deep Blue) | Deep Blue |
| **메인 제목** | 시스템 설계 방법론 (System Design Methodology) | Orbitron Bold, 크기: 50pt, 컬러: `#FFFFFF` |
| **서브 타이틀** | 수익 극대화 프레임워크 (Profit Maximization Framework) | Inter SemiBold, 크기: 30pt, 컬러: `#00FFFF` |
| **시각적 요소** | 제목 아래에 Neon Cyan으로 구성된 간단한 시스템 흐름 다이어그램의 '프레임' 형태를 배치. 이는 논리적 연결을 시각화함. | Neon Cyan Flow Lines |

#### B. 핵심 프로세스 슬라이드 (Process Breakdown Slide)

**목표:** 복잡한 과정을 단계별로 명확하게 제시하여 실행 가능성을 강조합니다.

| 요소 | 상세 사양 | 적용 스타일 |
| :--- | :--- | :--- |
| **레이아웃** | 3단 또는 4단 수직 분할 레이아웃. 각 단계는 좌측에 번호를, 우측에 내용을 배치. | 명확한 구조화 |
| **단계 제목 (Step)** | Step 1: 문제 정의 / Step 2: 모듈 설계 / Step 3: 시스템 구축 등 | Orbitron Bold, 크기: 24pt, 컬러: `#00FFFF` |
| **본문 내용** | 각 단계별 설명은 간결하게 작성하며, Deep Blue 배경 위에 흰색 텍스트를 사용. | Inter Regular, 크기: 18pt, 컬러: `#FFFFFF` |
| **연결선** | 각 단계 사이에는 Neon Cyan의 세로선(혹은 화살표)을 사용하여 논리적 연결성을 극대화함. | Neon Cyan Line Connector |
