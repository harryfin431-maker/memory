## Master System Blueprint 시각 에셋 구현 지침

확정된 Deep Blue/Neon 스타일 가이드라인과 Master System Blueprint를 기반으로 Figma에서 즉시 구현할 수 있는 구체적인 레이아웃 와이어프레임 및 디자인 시스템 적용 지침을 아래와 같이 제공합니다. 이 지침은 모든 시각 요소가 '시스템 설계자' 포지셔닝과 신뢰감을 극대화하도록 정의됩니다.

### 1. 디자인 시스템 재확인 (Deep Blue/Neon Style Guide)

| 요소 | 값 | 설명 |
| :--- | :--- | :--- |
| **Primary Color** | `#0A1931` (Deep Blue) | 배경, 주요 구조체에 사용되어 신뢰감과 깊이를 부여합니다. |
| **Accent Color 1** | `Electric Purple` (`#B162FF` 또는 유사) | 시스템 흐름, 핵심 강조 요소에 사용됩니다. |
| **Accent Color 2** | `Neon Cyan` (`#00FFFF` 또는 유사) | 데이터 흐름, 논리적 연결성, 미래지향적인 느낌을 시각화합니다. |
| **Typography (Heading)** | *Inter/Montserrat Bold* | 시스템의 권위와 명확성을 강조합니다. |
| **Typography (Body)** | *Inter/Roboto Regular* | 가독성과 정보 전달에 중점을 둡니다. |
| **Visual Theme** | Dark Mode 기반, 네온 하이라이트 | 어두운 배경 위에 밝고 선명한 색상을 사용하여 시스템의 논리적 연결성을 시각화합니다. |

### 2. Master System Blueprint 레이아웃 와이어프레임 (Figma 구현 기준)

**A. 인트로/Hero Section (시스템 포지셔닝 강조)**

*   **배경:** Primary Color (`#0A1931`) 전체 적용.
*   **메인 타이틀:** `System Designer` 또는 `Master System Blueprint` (Typography: Bold, Electric Purple).
*   **핵심 시각화 요소:** 시스템의 논리적 흐름을 나타내는 추상적인 노드/연결선 패턴을 Neon Cyan으로 구현하여 데이터 흐름과 연결성을 강조합니다.
*   **CTA 영역:** 중앙 하단에 배치하며, 배경색은 약간 밝은 Deep Blue 톤을 사용하여 대비를 줍니다.

**B. Methodology / Process Flow (시스템 프레임워크 시각화)**

*   **레이아웃:** 좌측 또는 상단에 프로세스 다이어그램 형식으로 배치.
*   **흐름 표현:** 각 단계(Step)는 Deep Blue 박스로, 다음 단계로의 연결선은 Neon Cyan 선으로 처리하여 논리적 순서를 명확히 합니다.
*   **요소 강조:** 핵심 변수나 결과값(KPIs)은 Electric Purple 텍스트와 아이콘을 사용하여 시각적으로 분리하고 강조합니다.

**C. Feature Block / Data Visualization (결과 및 증명)**

*   **기본 구조:** 카드(Card) 기반 레이아웃을 사용하며, 각 카드는 어두운 배경 위에 투명도가 적용된 Deep Blue 박스로 구성합니다.
*   **데이터 시각화:** ROI나 가치 점수 등 핵심 지표는 Neon Cyan의 선 그래프 또는 게이지 형태로 시각화하여 데이터 흐름을 극대화합니다.
*   **레이아웃 원칙:** 모든 블록 간의 간격(Spacing)은 균일하게 유지하며, 정보 밀도(Density)를 높여 프리미엄 느낌을 부여합니다.

### 3. 디자인 적용 가이드라인 요약

1.  **색상 사용 규칙:** Deep Blue는 안정성을, Electric Purple은 시스템의 핵심 논리를, Neon Cyan은 데이터 흐름과 미래지향성을 나타냅니다.
2.  **레이아웃 일관성:** 모든 레이아웃은 모바일/데스크톱 반응형을 기본으로 하며, 여백(Whitespace)을 적극적으로 사용하여 시각적 위계를 설정합니다.
3.  **텍스트 계층 구조:** 가장 중요한 정보(System Name, Core Metric)는 크기와 색상 대비를 최대화하여 강조하며, 세부 설명은 본문 폰트를 사용합니다.
