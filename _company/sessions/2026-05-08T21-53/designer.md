## 💎 최종 비주얼 시스템 가이드라인 (Asset Guide)

이 섹션은 모든 콘텐츠(썸네일, 영상 내 오버레이 등)에 적용될 **절대적인 규칙**입니다.

### 1. 컬러 팔레트 (Color Palette)

| 용도 | 색상 이름 | HEX 코드 | RGB | 사용 목적 |
| :--- | :--- | :--- | :--- | :--- |
| **Primary Background** | Deep Navy | `#0A192F` | `10, 25, 47` | 신뢰감, 안정성 제공 (기본 배경) |
| **Accent/Action** | Neon Cyan | `#00FFFF` | `0, 255, 255` | 성장, 수익성 강조 (KPI 및 핵심 데이터) |
| **Secondary Text** | Soft Gray | `#A0A0A0` | `160, 160, 160` | 보조 정보, 설명 텍스트 |
| **Highlight/Warning** | Deep Blue Accent | `#1E90FF` | `30, 144, 255` | 중간 단계, 비교 데이터 강조 |

### 2. 타이포그래피 시스템 (Typography System)

타이포그래피는 신뢰감과 미래지향성을 동시에 표현하도록 설계합니다.

| 요소 | 폰트 패밀리 | 스타일 | 크기 (Desktop 기준) | 사용 목적 |
| :--- | :--- | :--- | :--- | :--- |
| **Title (Headline)** | Montserrat | Bold, All Caps | 48px ~ 64px | 핵심 메시지 전달 (Neon Cyan 활용) |
| **Sub-Headline** | Montserrat | SemiBold | 24px ~ 32px | 부제 및 맥락 설명 |
| **Body Text / Data** | Roboto | Regular | 16px ~ 18px | 상세 정보, 설명, 데이터 표기 (Deep Navy 배경) |

### 3. 레이아웃 원칙 (Layout Principles)

*   **깊이(Depth):** Deep Navy를 사용하여 깊은 신뢰감을 형성하고, Neon Cyan을 사용하여 중요한 성과 지표(ROI)에 시선이 즉시 집중되도록 합니다.
*   **대비(Contrast):** 모든 데이터 오버레이는 배경 색상 대비가 극대화되어야 합니다. (Cyan 텍스트 vs Navy 배경)
*   **Flow:** 정보의 흐름은 항상 좌측 상단에서 우측 하단으로, 혹은 핵심 지표를 중심으로 시선이 자연스럽게 이동하도록 설계합니다.

## 📊 데이터 오버레이 표준 템플릿 프로토타입 (Data Overlay Prototype)

이 프로토타입은 영상 내 또는 썸네일 디자인에 적용될 **핵심 KPI 시각화 레이아웃**을 정의합니다.

### A. 핵심 지표 카드 (Key Metric Card: $Ad-ROI$ Focus)

**목표:** 가장 중요한 수익성 데이터를 Neon Cyan으로 압도적으로 부각시킵니다.

| 요소 | 위치/크기 (상대 좌표) | 스타일 적용 | 내용 예시 |
| :--- | :--- | :--- | :--- |
| **KPI 제목** | 좌측 상단 (X: 5%, Y: 5%) | Montserrat Bold, Neon Cyan (`#00FFFF`) | **$Ad-ROI$ Performance** |
| **핵심 수치** | 중앙 (X: 50%, Y: 30%) | **매우 크고 두꺼운 폰트**, Neon Cyan (`#00FFFF`), 배경은 Deep Navy의 반투명 박스 사용 | **+450%** (가장 큰 숫자) |
| **보조 수치** | 핵심 수치 바로 아래 (X: 50%, Y: 35%) | Roboto Regular, Soft Gray (`#A0A0A0`) | $LTV$ 대비 $\text{CAC}$ 효율성 분석 |
| **Context Bar** | 하단 전체 영역 (Y: 70%~) | Deep Navy 배경에 Neon Cyan 테두리 박스 | "최근 30일 평균" 또는 "목표 달성률" 표시 |

### B. 프로세스 흐름 다이어그램 (Process Flow Diagram Template)

**목표:** $Ad-ROI$ 프레임워크의 논리적 흐름을 시각적으로 명확하게 제시합니다. (Deep Navy/Neon Cyan 스타일 적용)

| 단계 구분 | 시각 요소 | 색상 및 스타일 | 설명 |
| :--- | :--- | :--- | :--- |
| **Phase 1: Input** | 사각형 노드 | Deep Navy 배경, Neon Cyan 테두리 | 데이터 수집 및 목표 설정 (Input Data) |
| **Phase 2: Process** | 화살표 연결 | Neon Cyan (흐름 강조) | $Ad-ROI$ 계산 로직 (Algorithm Flow) |
| **Phase 3: Output** | 원형 노드 | Neon Cyan 채우기, Deep Navy 테두리 | 최종 결과 도출 ($Ad-ROI$ Result) |

**레이아웃 좌표 예시:**

*   전체 다이어그램 영역 크기: 가로 100%, 세로 60%
*   Phase 1 (Input): (X: 20%, Y: 15%)
*   Phase 2 (Process): (X: 50%, Y: 30%) - 중앙에 위치하여 흐름 강조
*   Phase 3 (Output): (X: 80%, Y: 45%)
