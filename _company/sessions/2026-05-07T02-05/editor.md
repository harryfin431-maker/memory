## 영상 편집 디렉션 및 B-roll 전략 구체화

Designer가 구현한 Master System Blueprint와 Scene-by-Scene 지침을 기반으로, 실제 영상 소스에 적용할 구체적인 컷 리듬(Pacing)과 B-roll 매칭 전략을 시간 코드와 함께 제시합니다. 이 지침은 **'시스템 설계자' 포지셔닝**을 극대화하고, Deep Blue/Neon 스타일 가이드라인을 일관되게 유지하는 데 중점을 둡니다.

---

### 🎬 영상 개요 및 목표 리듬 설정

*   **전체 톤:** 논리적이고, 데이터 기반이며, 미래 지향적인 '시스템 설계자'의 권위 강조.
*   **컷 리듬 (Pacing):** 핵심 정보 전달 구간(데이터 제시)에서는 **2~3초 단위**의 빠른 전환을 사용하고, 개념 설명 및 B-roll 삽입 시에는 **4~5초**를 할애하여 시청자가 정보를 소화할 시간을 부여합니다.
*   **전환 효과:** 모든 컷 전환 시점에 Designer가 정의한 **네온 스캔 라인 애니메이션**을 적용하여 논리적 흐름의 연결성을 강조합니다.

### ⏱️ Scene-by-Scene 편집 지침 (예시)

*(실제 영상 소스에 맞춰 아래 시간 코드를 채워 넣으십시오. 아래는 Blueprint 구조를 따른 예시입니다.)*

#### A. 인트로 화면 / 후크 시퀀스 (0:00 - 0:15)
**목표:** 시청자의 즉각적인 몰입 및 '시스템 설계자'의 권위 확립.
| 시간 코드 | 영상 소스 내용 | 컷 길이 | 편집 액션 (Editor Directive) | B-roll/화면 효과 |
| :--- | :--- | :--- | :--- | :--- |
| **0:00 - 0:03** | 메인 타이틀 등장 ("시스템 설계자") | 3초 | 빠르게 나타나며 네온 블로우 효과 적용. | Deep Blue 배경, Cyan/Purple 라인 애니메이션 시작. |
| **0:03 - 0:08** | 서브 카피 제시 (수익성 시각화) | 5초 | 타이틀이 안정화되는 동안 Subtle Glow Effect 유지. | Neon Cyan & Electric Purple 선으로 데이터 흐름 강조. |
| **0:08 - 0:15** | 시스템 전체 흐름 시각화 (Master Blueprint 등장) | 7초 | 좌측에서 우측으로 스캔 라인 애니메이션이 화면을 가로지르며 Master Blueprint를 드러냄. | 모든 정보는 네온 색상으로 연결되어 논리적 순서를 시각적으로 증명. |

#### B. 핵심 시스템 프레임워크 (Methodology Flow)
**목표:** Input -> Process -> Output의 논리적 관계를 명확하고 빠르게 전달.
| 시간 코드 | 영상 소스 내용 | 컷 길이 | 편집 액션 (Editor Directive) | B-roll/화면 효과 |
| :--- | :--- | :--- | :--- | :--- |
| **0:15 - 0:30** | Step 1: Input (데이터 수집 및 검증) | 6초 | 데이터 입력 과정의 역동적인 움직임을 강조. | Input 박스에 Neon Cyan으로 포커스, 배경은 Deep Blue 유지. |
| **0:30 - 0:45** | Step 2: Process (논리적 흐름 구축) | 15초 | 복잡한 데이터가 논리적으로 연결되는 과정을 빠르게 전환하여 시각화. | Electric Purple 라인이 Input 데이터를 처리하며 흐르는 모습 강조. |
| **0:45 - 1:00** | Step 3: Output (결과 도출 및 검증) | 15초 | 최종 결과물이 명확하게 제시되는 클로즈업 전환. | 모든 연결선이 안정적으로 수렴하며, 최종 결과가 Cyan으로 강조됨. |

| :--- | :--- | :--- | :--- | :--- |

### 🖼️ B-roll 전략 제안 (Data & Logic Focus)

B-roll은 추상적인 개념보다는 **'시스템', '데이터 흐름', '분석', '정확성'**을 시각화하는 데 집중합니다.

1.  **데이터 시각화:** 복잡한 데이터가 그래프나 차트 형태로 실시간으로 변환되는 모션 그래픽 클립 (Neon Cyan/Purple 강조).
2.  **시스템 설계:** 복잡한 회로도, 논리적 연결망, 코드 라인 등이 빠르게 스크롤되거나 연결되는 시퀀스.
3.  **분석 및 검증:** 데이터 포인트에 마우스를 대고 분석하는 듯한 클로즈업 샷 (Focus/Zoom 효과 활용).
4.  **환경:** 배경은 항상 Deep Blue를 유지하되, 강조 요소(텍스트, 라인)만 네온 색상을 사용하여 **데이터의 논리적 흐름을 시각화**합니다.
