## 🎬 최종 비주얼 실행 블루프린트 (Video Production Ready)

본 문서는 Writer의 스크립트와 Developer의 데이터 구조를 기반으로, 영상 제작 단계에서 즉시 적용 가능한 모든 시각화 리소스 및 디자인 가이드를 최종 확정합니다.

### 1. 브랜드 시스템 최종 확정 (Visual System Lock)

| 요소 | 정의 | 상세 값 (실행 기준) |
| :--- | :--- | :--- |
| **배경 색상** | Deep Navy (신뢰감/깊이) | `#0A192F` |
| **강조 색상** | Neon Cyan (미래지향/에너지) | `#00FFFF` |
| **보조 텍스트** | Light Gray / White | `#E6E6FA` (주요 데이터), `#FFFFFF` (제목) |
| **타이포그래피** | 시스템 설계자 테마 | **Headline:** Montserrat Bold, Neon Cyan / **Body:** Roboto Regular, White |
| **리듬 규칙** | 3~5초 컷 리듬 | 모든 장면 전환 및 핵심 데이터 강조는 최대 5초를 넘기지 않도록 설정. (빠른 시각적 흐름 유지) |

### 2. 스크립트 기반 시퀀스별 비주얼 디렉션 (Sequence-by-Sequence Visual Direction)

Writer가 제공한 스크립트(Hook 1 선택 기준)에 맞춰, 데이터 구조와 연동하여 각 구간의 시각적 목표를 구체화합니다.

| 시간대 | 내레이션 핵심 내용 | 시각적 목표 (Visual Focus) | 색상/효과 적용 (Deep Navy / Neon Cyan) | 데이터 연동 지침 (Developer Data Link) |
| :--- | :--- | :--- | :--- | :--- |
| **0:00-0:05** | 후크 도입부 ("당신의 시스템이 돈을 벌고 있습니까?") | 복잡한 시스템 다이어그램의 빠른 움직임 (Motion Graphics). 데이터가 폭발하며 시각화. | 배경 Deep Navy, 데이터 라인 Neon Cyan으로 빠르게 스캔. | LTV/CAC 비율의 극단적인 낮은 값(예: 0.3)을 중앙에 크게 노출. |
| **0:05-0:30** | 문제 제기 (비효율성 지적) | 비효율성을 상징하는 복잡하고 엉킨 노드 다이어그램이 느려지며 정체되는 모습. | 전체적으로 차분한 Deep Navy, 오류/정체 구간에만 미세한 Cyan 깜빡임 적용. | 기존 자동화 프로세스(Workflow)의 비효율적인 연결을 시각화. |
| **0:30-1:15** | 핵심 솔루션 제시 (시스템 설계자 역할) | 복잡했던 노드들이 깔끔하고 논리적인 모듈로 분해되어, 중앙의 '수익 설계 시스템'으로 명확하게 재구성되는 과정. | 모든 프로세스가 Neon Cyan 선으로 연결되며 모듈화됨. 다이어그램이 3D 공간에서 입체적으로 정리되는 효과. | LTV/CAC KPI 구조(LTV, CAC, ROI)가 명확한 박스로 분리되어 시각화됨. |
| **1:15-2:00** | 방법론 소개 (MVP 핵심) | 세 단계(진단 $\rightarrow$ 설계 $\rightarrow$ 실행)를 명확하게 보여주는 3단계 프로세스 흐름도. 각 단계별로 결과물(시스템, KPI)이 빛나는 효과. | 각 단계는 독립된 Cyan 박스로 강조. 최종 결과는 가장 밝은 Neon Cyan으로 폭발하며 마무리. | 데이터 모델의 논리적 흐름을 따라 세 단계를 명시적으로 연결. |

### 3. 썸네일 컨셉 검토 및 권장안 (Thumbnail Concept Review)

이전 작업에서 제시된 3가지 컨셉을 영상 내용(시스템 설계, 수익 극대화)에 맞춰 최종 평가하고 실행 가능한 디자인 가이드로 확정합니다.

| 컨셉 | 핵심 메시지 | 추천 이유 | 적용 가이드라인 |
| :--- | :--- | :--- | :--- |
| **컨셉 1: 결과 중심 폭발형 (선택 권장)** | 비효율 $\rightarrow$ 수익 극대화의 명확한 대비. | 후크와 메시지(ROI)에 가장 직접적으로 연결됨. 시청자의 즉각적인 클릭 유도력이 높음. | 배경은 Deep Navy. 중앙에 거대한 Neon Cyan으로 강조된 '1:10 미만' 또는 '시스템 설계자' 텍스트 배치. 인물/아이콘은 최소화하고 데이터 위주로 구성. |
| **컨셉 2: 심리적 딜레마형** | "노력 대비 결과 부재"라는 감정적 공감 유도. | 문제 제기 단계(0:05-0:30)에 적합하며, 시청자의 페인 포인트에 깊이 침투함. | 어두운 배경 위에 인물의 복잡한 표정이나 혼란을 상징하는 다이어그램을 배치하여 감정을 전달. Cyan은 불안정함을 표현. |
| **컨셉 3: 권위/시스템 강조형** | '설계자'라는 전문가적 포지셔닝 강조. | 채널의 전문성과 시스템에 대한 신뢰도를 구축하는 데 유리함. | 깔끔한 레이아웃, 타이포그래피 중심. Deep Navy 배경 위에 정돈된 다이어그램과 명확한 구조만을 배치하여 권위를 부여. |
