## 🎬 최종 비주얼 에셋 패키지 및 디자인 가이드라인 확정

영상 제작팀(Editor)이 즉시 사용 가능하도록, 앞서 수립된 브랜드 시스템과 $Ad-ROI$ 시각화 모듈에 대한 최종 지침을 정리하여 제공합니다. 이 문서는 모든 영상 콘텐츠의 시각적 기준점(Single Source of Truth)이 됩니다.

### 1. 브랜드 비주얼 시스템 확정 (The Foundation)

**🎨 핵심 컬러 팔레트:**
*   **Primary Color (신뢰/깊이):** Deep Navy (#0A1931) — 배경, 주요 텍스트, 안정감을 담당합니다.
*   **Accent Color (혁신/수익):** Neon Cyan (#00FFFF) — 데이터 흐름, 성장 엔진 모듈 연결, 강조 효과에 사용됩니다.
*   **Neutral Color (가독성):** Off-White/Light Gray (#F5F5F5) — 본문 텍스트 및 배경으로 사용되어 대비를 극대화합니다.

**🖋️ 타이포그래피 시스템:**
*   **Headline (제목/강조):** Montserrat Bold 또는 Pretendard Black (Neon Cyan 강조 시).
*   **Body (본문/설명):** Pretendard Regular (Deep Navy 배경 위에서 높은 가독성 확보).

### 2. $Ad-ROI$ 성장 엔진 시각화 모듈 상세 지침

$Ad-ROI$ 변환 모듈은 **'성장 엔진(Growth Engine)'** 형태로 디자인되며, 논리적 흐름을 극대화하는 데 중점을 둡니다.

| 구성 요소 | 역할 및 시각화 방식 | 적용 컬러/스타일 |
| :--- | :--- | :--- |
| **Input Module (투입)** | LTV/CAC 데이터 입력 지점. 현실의 문제를 제시함. | Deep Navy 배경, White 텍스트 |
| **Transformation Core (변환 엔진)** | $Ad-ROI$ 공식이 작동하는 핵심 과정. 복잡한 계산을 단순화하여 보여줌. | 전체 모듈 테두리: Neon Cyan |
| **Output Module (결과)** | 최종 수익 지표($Ad-ROI$)가 시각적으로 폭발하며 제시되는 결과 영역. | Neon Cyan으로 채워진 그래프/게이지 형태 |
| **Connection Flow** | Input → Transformation → Output의 논리적 흐름을 연결하는 선 또는 화살표. | Neon Cyan (선 자체) |

**💡 디자인 원칙:** 모듈 간의 연결부는 반드시 **Neon Cyan**으로 처리하여, 시청자가 데이터가 어떻게 '성장 엔진'을 통해 수익으로 변환되는지 즉각적으로 이해하도록 유도합니다. 전체 배경은 **Deep Navy**로 설정하여 고급스럽고 신뢰감 있는 분위기를 유지합니다.

### 3. 최종 비주얼 에셋 가이드라인 (Asset Guide for Editor)

Editor가 바로 적용할 수 있도록 핵심 디자인 요소를 정리합니다.

| 요소 | 지침 | 세부 값/적용 방법 |
| :--- | :--- | :--- |
| **전체 배경** | Deep Navy (#0A1931) 사용을 기본으로 설정. | 모든 영상의 메인 톤앤매너 확립. |
| **강조 효과** | Neon Cyan (#00FFFF)은 오직 데이터 흐름과 핵심 결과에만 사용. | 그래프, 모듈 경계선, ROI 수치 강조. |
| **폰트 설정** | Headline: Montserrat Bold / Body: Pretendard Regular | 모든 텍스트에 일관성 유지. |
| **데이터 시각화** | LTV/CAC 게이지는 반드시 '안정(Navy) → 최적(Cyan) → 위험(Red Accent)'의 3단계로 명확히 구분하여 표현할 것. |
