# 🎬 최종 비주얼 시스템 및 영상 에셋 준비 체크리스트

본 체크리스트는 확정된 **Deep Navy/Neon Cyan 시스템**과 **$Ad-ROI$ 시각화 레이아웃**의 일관성을 최종 점검하고, 모든 영상 제작에 필요한 에셋(템플릿, 가이드라인)이 완벽히 준비되었는지 확인하기 위해 설계되었습니다.

## 1. 글로벌 시스템 일관성 검증 (Consistency Check)

| 항목 | 세부 내용 | 상태 (Y/N/NA) | 점검 사항 |
| :--- | :--- | :--- | :--- |
| **컬러 팔레트** | Deep Navy ($#0A1931$) 및 Neon Cyan ($#00FFFF$)의 사용 비율이 전체 디자인에 일관되게 적용되었는가? | | 모든 배경, 강조색, 보조색에서 색상 대비와 심리적 효과가 통일되는지 확인. |
| **타이포그래피** | 확정된 폰트 계층 구조(Header, Subtitle, Body, Data Label)가 전체 시스템에 적용되었는가? (예: Poppins/Montserrat 기반) | | 모든 크기 및 굵기가 브랜드의 신뢰감과 미래지향성을 반영하는지 확인. |
| **레이아웃 원칙** | Hero Section 와이어프레임에서 정의된 그리드 및 여백(Padding/Margin) 규칙이 모든 콘텐츠에 적용되었는가? | | 모바일, 데스크톱, 유튜브 썸네일 등 다양한 매체 비율에 따른 레이아웃 변형 규칙 확인. |
| **$Ad-ROI$ 시각화** | ROI 지표를 표현하는 차트(Bar/Line/Gauge)의 색상과 스타일이 Deep Navy/Neon Cyan 테마 내에서 명확하게 구분되는가? | | KPI 수치와 비주얼 요소 간의 연결성이 완벽한지 확인. |

## 2. 영상 제작 에셋 준비 상태 (Asset Readiness Check)

| 항목 | 세부 내용 | 상태 (Y/N/NA) | 조치 사항 |
| :--- | :--- | :--- | :--- |
| **썸네일 템플릿 (3안)** | 컨셉 3안 중 최종 선택된 디자인에 대해, 유튜브 규격(16:9)에 최적화된 사이즈 및 안전 영역 설정이 완료되었는가? | | 각 썸네일의 핵심 메시지 영역($50\% \sim 70\%$ 구역)에 대비되는 색상 배치가 명확한지 확인. |
| **인트로/아웃트로 템플릿** | 브랜드 아이덴티티(컬러, 로고 위치, 타이포그래피 스타일)가 포함된 표준 인트로 및 아웃트로 애니메이션 템플릿이 디자인되었는가? | | 움직이는 요소(Motion Graphics)의 속도와 색상 변화가 시스템을 따르는지 검토. |
| **데이터 오버레이 가이드** | $Ad-ROI$ 데이터를 영상에 삽입할 때, 배경과의 대비를 최대화하는 텍스트 박스 및 그래프 레이아웃 표준이 정의되었는가? | | 데이터 시각화(차트)와 브랜드 디자인의 충돌을 방지하기 위한 구체적인 좌표/스타일 매뉴얼 준비. |
| **브랜딩 가이드** | 영상에 사용될 최종 타이포그래피 세트 및 색상 코드를 포함한 '영상용 디자인 가이드라인' 문서가 완성되었는가? | | 에디터가 별도 질문 없이 바로 작업에 착수할 수 있도록 모든 참조 정보를 한 곳에 모음. |

## 3. 최종 검토 및 승인 (Final Approval)

1.  **일관성 점검:** 모든 준비된 에셋(템플릿, 가이드라인)을 Deep Navy/Neon Cyan 시스템의 **'신뢰감 + 미래지향적'**이라는 핵심 정체성에 비추어 3회 이상 시각적으로 검토한다.
2.  **실행 가능성 확인:** 영상 제작 담당자가 이 체크리스트만 보고 디자인 에셋을 수정하거나 새로 만들 필요가 없음을 확인한다. (Self-Correction: 모든 수치는 **구체적인 코드 및 좌표**로 명시되어야 함.)
