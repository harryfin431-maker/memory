## 🎬 최종 비주얼 리소스 패키지 전달 및 최적화 (Module_Upgrade 준비 완료)

Editor의 즉각적인 영상 제작 착수를 위해, 확정된 **Deep Navy/Neon Cyan 시스템**을 기반으로 모든 시각 자료를 편집 단계에 맞게 최적화하고 일관성을 최종 보증합니다. 이 패키지는 'ROI 극대화 게이지'의 시각적 흐름을 중심으로 구성되었습니다.

### 1. 디자인 시스템 최종 확정 (Visual System Lock)

| 요소 | 상세 정의 | 적용 코드/스타일 | 용도 |
| :--- | :--- | :--- | :--- |
| **주요 배경색** | Deep Navy (신뢰감, 깊이) | `#0A192F` | 전체 화면 배경 및 주요 구조 프레임 |
| **강조 색상** | Neon Cyan (에너지, KPI) | `#00FFFF` | ROI 게이지 채움, 핵심 지표 강조, CTA 버튼 |
| **보조 텍스트** | Off-White/Light Gray | `#E0F7FA` | 본문 및 설명 텍스트 |
| **주요 타이포그래피 (헤드라인)** | Sans-Serif (미래지향적) | Pretendard Bold / Montserrat ExtraBold | 핵심 메시지, KPI 수치 강조 |
| **보조 타이포그래피 (본문)** | Sans-Serif (가독성) | Pretendard Regular / Roboto | 상세 설명 및 프레임워크 설명 |

### 2. 핵심 비주얼 에셋 목록 및 최적화 버전

#### A. ROI 극대화 게이지 목업 (핵심 시각 요소)

**목표:** LTV/CAC 비율을 중심으로 수익성(Cyan)과 안정성(Navy)의 관계를 즉시 이해하도록 함.

*   **Asset 1: ROI 게이지 메인 컴포넌트**
    *   **구조:** 원형 진행 게이지 (Deep Navy 배경).
    *   **채움 방식:** Neon Cyan이 왼쪽에서 오른쪽으로 채워짐.
    *   **최적화:** 게이지의 바늘(Pointer)은 Neon Cyan으로 처리하며, 현재 수치(예: 45% 또는 실제 KPI 값)가 가장 두드러지게 표시되도록 디자인함. (모션 시작점/종료점을 명확히 정의하여 편집 시 애니메이션 적용 용이하게 함.)
*   **Asset 2: LTV/CAC 비율 비교 레이아웃**
    *   **구조:** 양분된 분할 화면 (Deep Navy 배경). 왼쪽은 'LTV' (안정성), 오른쪽은 'CAC' (비용).
    *   **시각화:** 두 지표 사이에 명확한 시각적 경계선(Neon Cyan)을 설정하여, 시스템 설계의 균형점을 강조함.

#### B. 브랜드 타이포그래피 스타일 가이드라인 (Text Style Guide)

*   **헤드라인 폰트 사용 예시:**
    *   `ROI MAXIMIZATION` : Pretendard Bold, 크기 64px, 색상 `#00FFFF`
*   **본문 폰트 사용 예시:**
    *   `The system is designed to maximize returns.` : Pretendard Regular, 크기 24px, 색상 `#E0F7FA`

#### C. 프로세스 다이어그램 템플릿 (시스템 구조화)

*   **스타일:** Flowchart 기반의 모션 그래픽용 벡터 템플릿.
*   **색상 매핑:**
    *   **Process Blocks (단계):** Deep Navy 테두리, 내부 채움은 연한 Cyan 그라데이션.
    *   **Connectors (흐름선):** Neon Cyan으로 강조하여 데이터의 흐름을 시각적으로 연결함.
*   **최적화:** 이 템플릿은 **Editor가 직접 모션 그래픽으로 구현하기 쉽도록** 각 단계별 박스(Block)와 화살표(Flow)만 벡터 형태로 분리하여 제공합니다.
