# 웹사이트 MVP 나머지 섹션 디자인 시안 (Methodology & Feature Block)

제공된 시스템 프레임워크와 Deep Blue/Neon 비주얼 가이드라인을 완벽하게 적용하여, 웹사이트의 핵심인 **'Methodology(방법론)'** 및 **'Feature Block(핵심 기능 블록)'** 섹션에 대한 구체적인 디자인 시안을 제시합니다.

---

## 1. Methodology (시스템 설계 방법론) 섹션 디자인 시안

이 섹션은 '시스템 설계자'로서의 권위와 체계성을 시각적으로 전달하는 데 중점을 둡니다.

**[레이아웃 컨셉]**
*   **배경:** Deep Blue (#0D1B2A)를 주 배경으로 사용하며, 깊이감을 부여합니다.
*   **강조 요소:** Neon Cyan (#00FFFF)을 핵심 프로세스 흐름과 각 단계의 중요 포인트를 강조하는 데 사용합니다.

**[디자인 상세 스펙]**

| 구성 요소 | 디자인 적용 (Deep Blue/Neon Cyan 기반) | 설명 및 목적 |
| :--- | :--- | :--- |
| **섹션 제목** | **Methodology: 시스템 설계 방법론** (폰트: Montserrat Bold, 색상: Neon Cyan) | 섹션의 주제를 명확히 제시하고 시각적 주의를 집중시킵니다. |
| **전체 레이아웃** | 3단계 수평 프로세스 또는 4단계 순차적 흐름 다이어그램 | 복잡한 시스템을 단순화하여 '체계적인 접근'을 강조합니다. |
| **단계별 디자인 (Step Flow)** | 각 단계는 좌측에 큰 아이콘(Neon Cyan)과 우측에 설명 박스로 구성. | 구조적이고 논리적인 흐름을 시각적으로 연결합니다. |
| **단계 1: 시스템적 사고** | 배경은 어두운 블루, 프로세스 화살표는 Neon Cyan으로 강조. (핵심 키워드: 전체 흐름 파악) | 문제 인식 단계임을 명시하며 시작점을 설정합니다. |
| **단계 2: KPI 기반 설계** | 박스 배경을 미세하게 밝은 블루(Light Blue/Gray)로 처리하여 다음 단계와의 차이를 줍니다. (핵심 키워드: 측정 가능한 목표 설정) | 구체적인 방법론임을 강조하며 실행의 기준을 제시합니다. |
| **단계 3: 자동화된 프레임워크** | 가장 밝은 Neon Cyan으로 박스 테두리를 강조하고, 중앙에 'Framework' 아이콘 배치. (핵심 키워드: 시스템 구축 및 자동화) | 최종 결과물(시스템)과 실행의 완성도를 시각적으로 극대화합니다. |
| **폰트 스타일** | 제목은 산세리프 계열의 굵은 서체, 본문은 가독성이 높은 얇은 서체를 사용 (Deep Blue 배경 위에서 대비 강조). |

---

## 2. Feature Block (핵심 기능 블록) 섹션 디자인 시안

이 섹션은 제공하는 시스템의 구체적인 '결과물' 또는 '모듈'을 제시하여 가치를 입증합니다.

**[레이아웃 컨셉]**
*   **배경:** Deep Blue 배경 유지.
*   **구성:** 모듈별로 카드(Card) 형태로 분리하여, 각 기능이 독립적인 시스템임을 강조합니다.

**[디자인 상세 스펙]**

| 구성 요소 | 디자인 적용 (Deep Blue/Neon Cyan 기반) | 설명 및 목적 |
| :--- | :--- | :--- |
| **섹션 제목** | **Module: 구현 가능한 핵심 모듈** (폰트: Montserrat Bold, 색상: Neon Cyan) | 제공되는 산출물이 구체적인 '모듈'임을 명시합니다. |
| **카드 레이아웃** | 3~4개의 기능 블록을 그리드(Grid) 형태로 배치. 각 카드는 Deep Blue 배경에 약간의 투명한 블루 그라데이션을 적용하여 깊이감을 부여합니다. | 모듈 간의 시각적 분리를 통해 복잡성을 관리하고 가독성을 높입니다. |
| **모듈 1: KPI 측정 엔진** | 아이콘: 목표/차트 (Neon Cyan). 제목: '측정 가능한 설계' | 시스템의 핵심 결과물인 '측정 가능성'을 강조합니다. |
| **모듈 2: 프로세스 자동화 모듈** | 아이콘: 기어/자동화 (Neon Cyan). 제목: '반복 없는 실행 프레임워크' | 시간 절약과 자동화라는 시스템의 가장 큰 이점을 부각합니다. |
| **모듈 3: 비주얼 아키텍처** | 아이콘: 구조도/Blueprint (Neon Cyan). 제목: 'Deep Blue/Neon 비주얼 시스템' | 브랜드 디자인 가이드라인이 포함되어 있음을 명시하며, 시각적 자산의 가치를 강조합니다. |
| **상호작용 효과** | 마우스를 올리면(Hover) 카드 배경이 Neon Cyan으로 미세하게 밝아지며, 카드 내부 텍스트가 Deep Blue에서 더욱 선명하게 대비되도록 설정. | 사용자의 탐색 행동에 즉각적이고 활발한 피드백을 제공하여 몰입도를 높입니다. |
